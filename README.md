# PruebasC
Esta es una recopilación de mis proyectos de práctica en C#, donde he estructurado diferentes soluciones para dominar desde los fundamentos de la sintaxis hasta los conceptos avanzados de la Programación Orientada a Objetos (POO). A través de estos directorios, documento mi aprendizaje en el manejo de estructuras de control como ciclos y cadenas, la implementación de lógica aritmética en calculadoras y promedios, y el uso crítico de herramientas de robustez como el manejo de excepciones con try-catch. Además, profundizo en la arquitectura de software mediante el uso de herencia, interfaces, composición y polimorfismo, utilizando funciones virtuales y modificadores de acceso para crear código escalable y organizado. Cada carpeta representa un módulo diseñado para resolver problemas específicos.
